linux常用命令笔记

# 前言

一些linux日常使用中的一些笔记

# 常用命令

  • unzip xxx.zip 解压指定文件夹
  • ls -a查看当前目录下所有文件
  • touch {xxx}查看指定文件
  • vi {xxx}编写指定文件
  • mkdir test创建test目录
  • rm -rf test删除test目录
    • -f：强制模式，不提示确认即删除文件或目录。
    • -i：交互模式，删除文件或目录前询问是否确认。
    • -r：递归模式，删除目录及其下所有文件和子目录。
    • -v：详细模式，显示删除的每个文件或目录名。
  • cd test进入test目录
  • mv test test2讲test目录名称改为test2

# pm2

node进程管理工具

# 安装

npm i -g pm2

这样安装后pm2可能还不是全局的，所以需要执行以下步骤:

执行：cd /usr/software/nodejs/bin
执行：ls 可以看到 pm2
执行：ln -s /usr/software/nodejs/bin/pm2 /usr/local/bin/
现在 pm2 命令也是全局了

# 常用命令

  • pm2 list 查看所有进程
  • pm2 restart {id} 重启指定id进程
  • pm2 stop {id}停止指定id进程
  • pm2 delete {id}删除指定id进程
