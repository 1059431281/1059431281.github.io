nuxt学习笔记

# 前言

一些日常工作的nuxt学习笔记

# asyncData

在服务端或者路由更替时执行的代码。返回的对象会赋值给data，如果data中以及会有对应的键存在，会被覆盖

# client-only

是一个组件，被包裹在内的，只会在客户端渲染（一般来说，一些服务器没有的api，或者会用到不适合在服务端请求获得的数据应当用该组件包裹起来）

# 路由中间件

nuxt的路由中间件（也称为路由导航）,可以在三处地方进行，执行顺序依次如下排序

  1. nuxt.config.js文件进行配置，该配置会应用到所有到所有路由上。

配置方法：

module.exports = {
  router: {
    middleware: 'stats' // 也可以是数组。这里填写stats，就是去会去匹配middleware文件夹下的stats文件
  }
}

# cookie-universal-nuxt

一款方便处理cookie的nuxt常见

# 安装

npm i cookie-universal-nuxt -S

在nuxt.config.js添加moudules:['cookie-universal-nuxt']

# 基础使用方法

因为是安装在全局的，所以可以直接：

设置cookie this.$cookies.set('token','test123')

获取cookiethis.$cookie.get('token')

清除cookiethis.$cookie.remove('token')
