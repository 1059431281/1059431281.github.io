    # 错误原因：

    ​ 在本地向远程仓库推送代码时报权限不足的错误，通常的原因就是本地权限不足，也就是 gitee/github 本地配置的账号不对

    # 解决方法：

    ​ 这里采用最简单方法，打开电脑的控制面板 → 用户账号 → 管理你的凭证 →windows 凭据 → 找到普通凭据里面对应网址编辑打开，修改成拥有对应仓库的账号密码
