Stable Diffusion尝鲜

# 起因

今年因为ChatGPT的火热，全国乃至全球都卷起一波AI潮，这里的AI绘画是最吸引人眼球的一项技术，毕竟画作可以应用到各种场景，商用的价值也很高。因为对AI绘画这一块颇感兴趣，我便开始了解这一块的技术。

# 啥是Stable Diffusion

维基百科介绍：

Stable Diffusion是2022年发布的深度学习文本到图像生成模型。它主要用于根据文本的描述产生详细图像，尽管它也可以应用于其他任务，如内补绘制、外补绘制，以及在提示词指导下产生图生图的翻译。

我们可以把它理解为一个算法，一个可以根据描述语生成图片的算法。

# Stable Diffusion Web UI

为了方便写作，下面把Stable Diffusion缩写为SD。

市面上有很多底层基于Stable Diffusion的软件或者网页，咱们程序员要是直接用这种玩意，未免有点太不极客了。我了解了一圈，发现有个叫Stable Diffusion Web UI的开源项目，实现了SD的算法，并提供了相对定制化的界面，让使用者可以进行DIY，绘制出自己的理想画作。那就让我用这个项目体验体验AI绘画。
